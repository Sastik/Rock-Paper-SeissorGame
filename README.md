# ✊✋✌️ Rock Paper Scissors Game

A fun and interactive **Rock Paper Scissors** web game where users can play against the computer.
This project is built using **HTML, CSS, and JavaScript**, focusing on beginner-friendly game logic and UI design.

---

## 🚀 Features

* 🎮 Play against computer (AI/random logic)
* ⚡ Instant result (Win / Lose / Draw)
* 🧠 Randomized computer choices
* 📊 Score tracking (if implemented)
* 📱 Responsive design (works on all devices)
* 🎨 Clean and simple UI

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## 🎯 How It Works

The game follows classic rules:

* Rock beats Scissors
* Paper beats Rock
* Scissors beats Paper
* Same choice = Draw

The computer randomly selects an option, and the result is calculated instantly.
